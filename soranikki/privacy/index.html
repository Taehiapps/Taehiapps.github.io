プライバシーポリシー

そらにっき · 最終改定日：2026年6月14日

Taehi Apps（以下「当社」）は、当社が提供する「そらにっき」（以下「本アプリ」）における個人情報等の取扱いについて、個人情報の保護に関する法律（個人情報保護法）その他の関連法令およびガイドラインを踏まえ、以下のとおりプライバシーポリシー（以下「本ポリシー」）を定めます。

1. 事業者情報

事業者名：Taehi Apps

所在地：日本国内（詳細はお問い合わせにより開示）

お問い合わせ：アプリ内「設定」→「お問い合わせ」 / taehee@taehi-apps.com

2. 取得する情報

当社は、本アプリの提供にあたり、以下の情報を取得します。

【A. アカウント・認証】

・匿名利用開始時に Supabase Auth により発行されるユーザー ID、セッショントークン（メールアドレス等の個人識別情報は取得しません）

・外部認証（Apple / Google）を利用した場合に提供される識別子、メールアドレス（提供される場合）、表示名（提供される場合）

【B. プロフィール・設定】

・ニックネーム、起床・振り返り通知の ON/OFF および通知時刻、登録場所の出入り通知の ON/OFF、ジオフェンス半径等の設定

・都道府県、年齢（ユーザーが任意で入力した場合のみ）

・オンボーディング完了状態、利用規約同意日時（terms_accepted_at）、最終アクティブ日時

【C. ユーザーコンテンツ】

・手動記録（テキスト、任意の位置情報、記録日時）

・記録に添付した写真（画像ファイル）

・登録場所（名称、緯度経度、半径、通知の ON/OFF）

・振り返りテキスト（日付単位）

・通知上の操作に基づく記録（例：起床確認「はい」、場所出入り確認「はい」、振り返り通知から入力されたテキスト）

【D. 位置情報】

・端末の GPS 等により取得される緯度・経度（場所の登録、出入り判定、記録への付与のため）

・当社サーバーには、常時の移動軌跡（位置履歴の連続ログ）は保存しません

【E. お問い合わせ】

・お問い合わせカテゴリ、本文、返信先メールアドレス（匿名利用時はフォーム入力、外部認証利用時はアカウントのメールアドレス（提供される場合））、アプリバージョン、プラットフォーム

【F. 端末内（当社サーバーに送信しない、または最小限のもの）】

・ジオフェンス判定用の端末内キャッシュ（登録場所の複製、内外状態、処理済み通知 ID 等）

・背景画像 URL の端末内キャッシュ、通知スヌーズ・却下状態、オンボーディング表示状態

・Web 版ログイン時のブラウザセッション（Cookie / ローカルストレージ）

3. 利用目的

取得した情報は、以下の目的で利用します。

（1）ユーザー認証、アカウント管理、利用規約同意の記録

（2）記録・写真・振り返り・登録場所の保存、表示、編集、削除、カレンダー等での閲覧

（3）登録場所の出入り検知、関連通知の送信、通知操作に基づく記録の作成

（4）起床・振り返り等のスケジュール通知の送信

（5）都道府県等に基づく天気表示、時間帯・天候に応じた背景表示、日替わりメッセージの表示

（6）お問い合わせへの対応、本人確認、重要なお知らせ

（7）不正利用の防止、障害対応、サービスの改善・新機能開発

（8）法令に基づく対応

目的外利用を行う場合は、法令に従い必要な手続を行います。

4. 第三者提供

当社は、次の場合を除き、個人データを第三者に提供しません。

（1）ユーザーの同意がある場合

（2）法令に基づく場合

（3）人の生命・身体・財産の保護のために必要で、本人同意が困難な場合

（4）公衆衛生の向上または児童の健全育成のために特に必要で、本人同意が困難な場合

（5）国の機関等への協力が必要で、本人同意により支障が生じるおそれがある場合

5. 委託・外部サービス（共同利用は行いません）

当社は、インフラ・認証・通知送信等のため、以下の外部サービスを利用します。必要な契約・安全管理措置を講じます。

【Supabase（Supabase Inc.）】認証、データベース（PostgreSQL）、オブジェクトストレージ（写真）、Edge Functions（お問い合わせ送信等）。データは当社が指定するリージョン（例：ap-northeast-1）に保存されます。

【Apple Inc. / Google LLC】Sign in with Apple、Google アカウントによる OAuth 認証。認証に必要な情報のみ Apple / Google とやり取りします。

【Resend（Resend Inc.）】お問い合わせメールの送信（Edge Function 経由）。

【Open-Meteo / wttr.in】天気情報の取得（都道府県等のおおよその位置に基づく気象データ。ユーザーの正確な GPS をこれらのサービスに送信することはありません）。

【Pexels / Wikimedia Commons 等】背景画像の取得（API 経由。ユーザー個人を特定する情報は送信しません）。

【Expo / EAS（開発・ビルド）】アプリのビルド・配信インフラ。エンドユーザー利用時に継続的に個人データを Expo に送信することはありません。

各サービスのプライバシーポリシーは、各事業者のサイトをご確認ください。

6. 位置情報の取扱い

1. 位置情報は、次の目的で利用します。（1）登録場所の作成・編集（2）登録場所の出入り判定（ジオフェンス）（3）ユーザーが許可した場合の記録への位置情報付与

2. 出入り判定をバックグラウンドで行う場合、OS の「常に許可」または同等のバックグラウンド位置情報権限が必要になることがあります。ユーザーは端末設定からいつでも OFF にできます。

3. 判定のため端末内で一時的に位置を取得することがありますが、当社サーバーに GPS の連続軌跡として保存することはありません。サーバーに保存されるのは、ユーザーが作成した記録・登録場所に紐づく座標等に限られます。

4. ジオフェンス判定の補助データ（内外状態、処理済み通知 ID 等）は、端末内ストレージ（AsyncStorage 等）に保存されることがあります。

7. 通知の取扱い

1. 本アプリは、ローカル通知（および OS が提供する通知拡張）を用いて、起床・振り返り・場所出入り等のプロンプトを表示します。

2. 通知上のボタン操作（例：「はい」）や、通知内テキスト入力（振り返り）の内容は、記録作成・保存のために当社サーバーに送信されることがあります。操作しなかった場合、記録は作成されません。

3. 通知を表示しないよう、端末の通知設定またはアプリ内設定から OFF にできます。

8. 写真・ストレージ

ユーザーが添付した写真は、Supabase Storage 上の当社管理バケットに保存されます。退会時には、関連する Storage オブジェクトも削除します（削除完了まで合理的な期間を要する場合があります）。

9. 保存期間

1. ユーザーコンテンツおよびアカウント情報は、退会（アカウント削除）まで保存します。

2. 退会後は、当社システムから削除します。バックアップ等に残存する場合は、合理的な期間内に消去します。

3. お問い合わせデータは、対応完了後、合理的な期間保存したのち削除します。

4. 法令により保存が義務付けられる場合は、当該期間保存します。

10. 安全管理措置

当社は、個人データの漏えい、滅失、毀損の防止のため、アクセス制御、通信の暗号化（HTTPS/TLS）、認証トークン管理、委託先の選定・監督等、合理的な安全管理措置を講じます。完全な安全を保証するものではありません。

11. ユーザーの権利（開示・訂正等）

ユーザーは、個人情報保護法に基づき、自己の個人データについて、利用目的の通知、開示、訂正・追加・削除、利用停止・消去、第三者提供の停止等を請求できます。

請求方法：アプリ内「設定」→「お問い合わせ」 または taehee@taehi-apps.com

本人確認のうえ、法令に従い対応します。退会機能により、ユーザー自身でデータ削除を行うこともできます。

12. 行動分析・広告

当社は、本アプリに第三者の行動分析・広告配信 SDK（例：Firebase Analytics、Adjust 等）を組み込んでいません。本アプリ内に第三者広告は表示しません（現バージョン）。

13. 未成年者

未成年者が本アプリを利用する場合、法定代理人の同意を得たうえで利用してください。未成年者から故意に個人情報を収集することはありません。

14. 本ポリシーの変更

当社は、法令改正、サービス内容の変更等に応じて本ポリシーを改定することがあります。重要な変更は、本アプリ内での告知等、合理的な方法で通知します。改定後の利用継続は、改定内容への同意とみなす場合があります。
